The basic Language of the Web: HTML

Posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three fundamental technilogiesL HTML, CSS and JavaScript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It's a markup language that web developers use to structure and descirbe the content of a webpage (not a pogramming language).

HTML consits of elements that descirbe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts: The opening tag The closing tag The actual element You can learn more at the MDN Web Docs.

  • To hand-craft beautiful websites instead of relying on tools like Wordpress or Wix To build web applications
  • To impress friends
